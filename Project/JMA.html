Japanese Morphological Analysis

Description: JMA is a Japanese morphological analyzer, which is used for chunking and morphology analysis. Also, there are many available Japanese morphological analyzers, like Chasen, Juman and Mecab. They are generic and result of morphology and chunks is not suitable for machine translation from Japanese to Chinese, JMA is designed for Japanese-Chinese machine translation tasks with the consideration of grain size. The JMA is referred to the dictionary and rules in Mecab.

the core algorithm is HMM with rules (the code is not public)
